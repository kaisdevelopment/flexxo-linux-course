# 🐧 Curso de Administração Linux — Flexxo (Caxias do Sul)

Repositório oficial dos materiais de apoio do curso de **Administração Linux** da escola Flexxo, polo Caxias do Sul.

---

## 🎯 Sobre o Curso

Este curso é ministrado em formato de **mentoria individual (1 a 1)**, com foco em formar administradores Linux prontos para o mercado corporativo de 2026.

O conteúdo vai além do operacional básico: cada aula conecta a **infraestrutura raiz** (terminal, permissões, processos, rede) com o pensamento de **automação, pipelines de dados e integração de sistemas** — competências essenciais para profissionais de infraestrutura moderna.

## 🧭 Filosofia do Curso

| Pilar | Descrição |
|-------|-----------|
| **Terminal First** | Todo conceito é praticado no terminal antes de qualquer interface gráfica |
| **Zero Trust** | Segurança não é módulo opcional — está presente em cada aula desde o dia 1 |
| **Pipeline Thinking** | A lógica de buscar, filtrar e agir é ensinada como fundamento para automação |
| **Mercado Real** | Exemplos e cenários baseados em data centers, provedores de nuvem e incidentes reais |
